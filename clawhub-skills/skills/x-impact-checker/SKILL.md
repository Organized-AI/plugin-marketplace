# X Impact Checker — Hack the Twitter Algorithm

## Purpose

Score tweets against the REAL open-source Twitter recommendation algorithm.
Twitter published their algorithm — this skill implements those exact scoring
factors.

## Algorithm Factors

| Factor | Optimal | Impact |
|--------|---------|--------|
| Character count | 70-150 chars | +20pts at sweet spot |
| Media presence | Image or video | +15pts (2x reach boost) |
| Engagement hook | Question at end | +10pts (drives replies) |
| Hashtag count | 0-2 | +10pts optimal, 3+ penalized |
| Engagement bait | Avoid "RT", "follow" | -15pts if detected |
| External links | Minimize | Reduces reach |

## Commands

| Command | Purpose |
|---------|---------|
| `score '<tweet>'` | Score 0-100 with factor breakdown |
| `rewrite '<tweet>'` | Algorithm-optimized rewrite suggestions |
| `analyze <@handle>` | Analyze account's top-performing patterns |

## Rewrite Rules

When suggesting rewrites:
1. Open with a hook (surprising stat, bold claim, or question)
2. Keep to 70-150 characters
3. End with a question to drive replies
4. Use 0-2 relevant hashtags max
5. Add media when possible
6. Never use: "RT", "follow", "retweet", "like this"
